# User APIs

## Register API : : POST
### /api/v1/users/new

## Login API : : POST
### /api/v1/users/login

## Logout API : : GET
### /api/v1/users/logout

## GetProfile API : : GET
### /api/v1/users/me

# Task APIs

## newTask API : : POST
### /api/v1/tasks/new

## getMyTask API : : GET
### /api/v1/tasks/all

## updateTask API : : PUT
### /api/v1/tasks/:id

## deleteTask API : : DELETE
### /api/v1/tasks/:id
